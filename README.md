# 🏨 VacNoti: Vacant Room Notification System

### 📌 Project Overview

VacNoti is a web-based application designed for hotel operations. It allows supervisors to instantly notify room attendants when a guest checks out, ensuring rooms are cleaned and ready for the next guest as quickly as possible.

### 🚀 Tech Stack

- **Language:** Python 3.x
- **Framework:** Streamlit (Web UI)
- **Environment:** Virtual Environment (`.venv`)

### 🛠️ How to Setup & Run

1. **Clone the repository:**
   `git clone https://github.com/alexyunkk/VanNoti.git`
2. **Create and Activate Virtual Environment:**
   `py -m venv .venv`
   `.venv\Scripts\activate`
3. **Install Dependencies:**
   `pip install -r requirements.txt`
4. **Run the Application:**
   `streamlit run VacNoti.py`

### 📋 Current Features

- **Supervisor Dashboard:** Simple interface to enter room numbers.
- **Notification Trigger:** Success messages once a room is dispatched.
- **Responsive Design:** Works on desktops and mobile browsers.
